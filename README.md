A simple python keylogger script that captures the owners keystroke as well as the associated 
window receiving those keystrokes. Uses third party libraries: -pynput and -pyGetWindow 
along with the python standard library. The user can press ESC to stop the keylogger, 
with a my_log.txt being instantiated in ``C:/users/currentuser`` or current working directory.
